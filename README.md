### Motivação 
Durante processo de realocação em projeto, promovido pela Zup, fui solicitado a apresentar projeto desenvlvido em  GoLang, onde já houvesse atuado.
Como os único projeto em GoLang onde atuei foi no Itau, não seria possível exibir o material desenvolvido, por causa da privacidade do Itau.
Decidi então, preparar uma API simples, para prova de capacidade.
